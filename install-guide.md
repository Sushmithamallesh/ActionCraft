# ActionCraft Setup

## Prerequisites

1. **ffmpeg** - Required for video processing
2. **Node.js** - v18 or higher
3. **pnpm** - For package management

## Installation Steps

### 1. Install ffmpeg

Choose your operating system:

**macOS:**
bash
brew install ffmpeg

**Windows:**
bash
choco install ffmpeg

**Linux:**
bash
sudo apt update && sudo apt install ffmpeg
